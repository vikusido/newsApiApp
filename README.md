# newsApiApp

Android-приложение для чтения новостей на Kotlin с Jetpack Compose. Новости загружаются через currentsapi.services.

## Возможности

- Лента последних новостей с подгрузкой при прокрутке
- Поиск по ключевым словам
- Открытие полной статьи через Custom Tabs
- Отображение заглушки если картинка недоступна

## Технологии

- Kotlin
- Jetpack Compose
- Retrofit и другие

## Запуск

Клонируйте репозиторий, откройте в Android Studio и запустите на устройстве или эмуляторе с Android 8.0+. Интернет-соединение обязательно.

## Примечания

Бесплатный тариф API отдаёт до 20 новостей за запрос. Часть статей приходит без изображений, скорее всего это особенность самого сервиса. Точность поиска также зависит от API.
